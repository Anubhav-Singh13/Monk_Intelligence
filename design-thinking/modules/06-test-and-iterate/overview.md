# Module 06 — Test & Iterate

**Days 22–25** · ~2.5 hours total

---

## What this module earns you

By the end of Day 25, you will be able to run a user test, synthesize what you observed into a decision, and make the judgment call that separates good practitioners from expensive ones: when to refine the solution, when to reframe the problem, and when to proceed. You will also consolidate everything from Days 12–21 in a retrieval session before adding the final layer.

The Test phase closes the loop that began in Empathize. The synthesis skill you built in Module 03 reappears here — because interpreting test observations is the same cognitive act as interpreting empathy research. By Day 24 you will feel that connection.
